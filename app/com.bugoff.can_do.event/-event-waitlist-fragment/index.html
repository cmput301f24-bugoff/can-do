EventWaitlistFragment

public class EventWaitlistFragment

Fragment for displaying and managing the waitlist of users for a specific event. Provides functionality to randomly select users from the waitlist and move them to the selected list. Displays a RecyclerView of waitlisted users, a progress bar during data load, and handles error messages.
