EventDetailsActivityOrganizer

public class EventDetailsActivityOrganizer

Activity to display detailed information about an event. This activity fetches and shows event details such as name, description, location, date, and image, along with a list of entrants. It also allows the user to perform actions such as joining or leaving the waiting list, viewing the event's watch list or selected list, sharing event details, and opening a map to the event's location.
